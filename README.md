# UAS_Pemogramanweb1
 Deskripsi Sistem Informasi Perpustakaan: BukuData Pro
1. Pendahuluan
BukuData Pro adalah platform manajemen inventaris perpustakaan berbasis web yang dirancang untuk mempermudah pustakawan dalam mengelola koleksi buku secara digital. Aplikasi ini mengedepankan antarmuka yang modern dengan gaya Glassmorphism untuk memberikan pengalaman pengguna (User Experience) yang bersih, mewah, dan interaktif.

2. Fitur Utama (Core Features)
Sistem ini mencakup seluruh siklus manajemen data buku yang krusial:

Autentikasi Keamanan: Sistem login terproteksi untuk memastikan hanya admin berwenang yang dapat mengakses dan mengubah data.

Dashboard Statistik: Tampilan ringkasan total koleksi dan kategori secara real-time untuk memantau status inventaris dalam sekejap.

Manajemen Data (CRUD): Fitur untuk menambah, melihat, memperbarui (edit), dan menghapus data buku dengan alur yang efisien melalui penggunaan modals.

Sistem Pelaporan Eksternal: Kemampuan untuk mengekspor data ke dalam format Excel dan PDF, memudahkan pelaporan resmi kepada pihak manajemen.

3. Teknologi yang Digunakan (Tech Stack)
Aplikasi ini dibangun menggunakan teknologi standar industri yang andal:

Bahasa Pemrograman: PHP (Server-side scripting).

Database: MySQL (Relational database) untuk penyimpanan data yang terstruktur.

Front-end Framework: Bootstrap 5 (Responsive Layout).

UI Library: SweetAlert2 untuk notifikasi interaktif dan Bootstrap Icons untuk navigasi visual.

Desain: Konsep Glassmorphism dengan perpaduan gradien warna pastel dan efek blur transparan.

4. Keunggulan Sistem
Keamanan Terpusat: Menggunakan fungsi cek_login() yang terpusat di file konfigurasi untuk mencegah akses ilegal melalui URL.
Efisiensi Input: Penggunaan modal pop-up untuk input data di halaman Dashboard mengurangi beban loading halaman dan mempercepat kerja admin.
